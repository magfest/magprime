Guidebook Tracks

We have two types of tracks for Guidebook schedule items: department tracks and content/warning/tag tracks. Many tracks are automatically added for panels, but you can add tracks to any event on the schedule. You can type in the Tracks field to search for different tracks. It may be helpful to have all events in one location start with a set of default tracks. To do this, click "Edit Location" on the schedule page, then add the tracks you want as defaults to the location. All new events created in that location will start with those tracks.

To request additional tracks, please contact the Guidebook team.
